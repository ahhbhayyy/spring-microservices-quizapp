# spring-microservices-quizapp

## 🚀 Tech Stack
- Spring Boot
- Spring Cloud
- Eureka Server
- API Gateway
- Hibernate / JPA
- PostgreSQL

## 🧩 Microservices
- Question Service
- Quiz Service


## ⚙️ Features

- REST APIs
- Service Discovery
- Load Balancing

2. Start all services
3. Run API Gateway
